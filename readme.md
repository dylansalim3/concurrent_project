## Quick Start Guide
Prerequisites:
- Java 8
- Gradle 5.2.1 (Optional)

To build the app, use gradle run.
``` 
cd PROJECT_FOLDER/concurrent_project 
gradle run 
```

To build the app without installing gradle.
``` 
cd PROJECT_FOLDER/concurrent_project
gradlew run 
```
